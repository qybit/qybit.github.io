第222场周赛

  2021-01-03   力扣 算法 周赛   贪心 数组 哈希表 双指针 二分法 0 0  评论 字数统计: 1774字 阅读时长: 4分

题单：

  • 卡车上的最大单元数
  • 大餐计数
  • 将数组分成三个子数组的方案数
  • 得到子序列的最少操作次数

T1：卡车上的最大单元数

方法一：贪心算法

贪心策略：我们优先装载单元数量最多的箱子。

如何证明贪心是最优解？

因为卡车只能装载 $truckSize$ 个箱子，也就是说我们只能选择固定数量的箱子，而每一个箱子装载的单元格数量是不同的，因此我们只要先装载单元格多的箱子，就可以装载最大总数的单元格。

代码

Java

class Solution {
    public int maximumUnits(int[][] boxTypes, int truckSize) {
        Arrays.sort(boxTypes, (o1, o2) -> o2[1] - o1[1]);
        int res = 0;
        for (int[] box : boxTypes) {
            if (truckSize < box[0]) {
                res += truckSize * box[1];
                break;
            }
            res +=  box[0] * box[1];
            truckSize -= box[0];
        }
        return res;
    } 
}

复杂度分析

时间复杂度：$O(NlogN)$

空间复杂度：$O(1)$

T2：大餐计数

开头

大餐的定义：数组中两个数的和恰好是 2 的幂。假设数组中存在两个数 2 和 6 ，它们下标不同，2 + 6 = 8 这就是一个大餐。

注意观察数据量，数组长度为 $10^5$ 且 数组中的值最大为 $2^{20}$。那么也就是说，两数之和的最大值为 $2^{21}$ 。

方法一：哈希表

假设在数组中要找出 $x + t = 2^k$ ，那么我们直接枚举 $[0, k], 0 \le k \le 21$ ，反过来寻找 $t = 2^k - x$ 。我们还需要统计每一个 $t$ 的出现次数，找到了就直接加上出现次数。

代码

c++

class Solution {
public:
    int countPairs(vector<int>& d) {
        unordered_map<int, int> hash;
        int mod = 1e9 + 7;
        int res = 0;
        for (int x : d) {
            for (int k = 0; k < 22; k++) {
                int t = (1 << k) - x;
                if (hash.count(t)) res = (res + hash[t]) % mod;
            }
            hash[x]++;
        }
        return res;
    }
};

复杂度分析

时间复杂度：$O(22*N)$

空间复杂度：$O(N)$

T3：将数组分成三个子数组的方案数

方法一：前缀和 + 双指针

开头

我们需要把数组划分为三个部分，分别为 $[0, left], [left+1, mid], [mid+1, right]$ ，使得 $Sum[0, left] <= Sum[left+1, mid] \ & \ Sum[left+1, mid] \le Sum[mid+1, right] $。$Sum$ 表示区间和。

在 快速求得连续区间的和，我们可以借助前缀和在 $O(1)$ 时间实现，剩下的事情就是如何划分区间的问题。

假设存在如下一种划分情况，区间 $A \le B \le C$

如下图，存在一种分割方案$A = Sum[0, x-1], B = Sum[x, i-1], C = Sum[i, n-1]$

此时是满足 $A \le B \le C$，对于某一个位置 $i$ 而言，存在一个合理的分割方案 $x$ ，我们可以找出 $x$ 的下界 $j$ 。换句话就是，$x$ 的最小范围取值可以是多少。对于 $k$ 也是同理，我们可以找出 $x$ 的最大范围可以取多少。

由于数组中都是非负整数，因此存在单调性。

不难发现，当 $j$ 向左边移动时， $A$ 是不断减少的，即 $B$ 是在不断增加的。我们要保证 $B \le C$。那么相反的，我们让 $j$ 不断向右移动，$B$ 是不断减少的，当 $B \le C$ 时我们就可以停止。

对于 $k$ 来说，当 $k$ 不断向右移动时，此时的 $A$ 是不断增加的，$B$ 是不断减小的，因此我们要保证 $A \le B$ 。

找出对于固定的 $i$ 来说，存在一个 $x$ ，我们只要找出其上下界 $[j, k]$ ，那么在 $i$ 贡献的分割方案为 $k - j + 1$。

代码

c++

class Solution {
public:
    int waysToSplit(vector<int>& nums) {
        int mod = 1e9 + 7;
        int res = 0;
        int n = nums.size();
        vector<int> s(n + 1, 0);
        for (int i = 1; i <= n; i++) s[i] = s[i-1] + nums[i-1];
        for (int i = 3, j = 2, k = 2; i <= n; i++) {
            // 如果 B > C 向右移动，减小 B
            while (s[n] - s[i-1] < s[i-1] - s[j-1]) j++;
            // 如果 A <= B 的情况下，k 最大能到哪个位置 
            while (k + 1 < i && s[k] <= s[i-1] - s[k]) k++;
            if (j <= k && s[k-1] <= s[i-1] - s[j-1] && s[i-1] - s[j-1] <= s[n] - s[i-1]) 
                res = (res + k - j + 1) % mod;
        }
        return res;
    }
};

复杂度分析

时间复杂度：$O(N)$

空间复杂度：$O(N)$

T4：得到子序列的最少操作次数

方法一：动态规划 + 二分

开头

在 $arr$ 中插入若干个整数后，可以在 $arr$ 中找到 $target$ 子序列。简而言之，就是去除掉 $target$ 和 $arr$ 的公共部分后，$target$ 中还剩下多少个元素的问题。

那么问题就可以转换为，$arr$ 和 $target$ 的最长公共子序列问题。

target = [5, 1, 3]
		  0  1  2
// arr 下标为 target 中的下标
arr = [5, 9, 1, 4, 2, 3, 4]
	   0  0  1	0  0  2  0

可以发现，$target$ 子序列的出现位置，在 $arr$ 中仍然是递增的特性。那么反过来，相应的，假如在 $target$ 中存在$arr$ 的子序列，$[1, 3]$ ，那么其对应下标在 $arr$ 中也仍然是递增的。

那么求两个数组的最长公共子序列，该算法的时间复杂度为 $O(N^2)$ 的，显然是不能通过代码。

定义：$q[i]$ 为长度为 $i$ 的最长上升子序列，$q[i]$ 是严格递增的。

对于 $target[i]$ ，如果出现在 $arr$ 中，则统计其下标。

那么对于任意一个 $a[i]$ 来说，我们在 $q[i]$ 中二分找出比 $a[i]$ 大的下标，如果不存在就把 $a[i]$ 加入末尾元素。

代码

c++

class Solution {
public:
    int minOperations(vector<int>& target, vector<int>& arr) {
        unordered_map<int, int> pos;
        for (int i = 0; i < target.size(); i++) pos[target[i]] = i;
        vector<int> a;
        for (int x : arr) {
            if (pos.count(x)) a.push_back(pos[x]);
        }
        // len 为最长上升子序列的长度
        int len = 0;
        int n = a.size();
        vector<int> q(n + 1);
        for (int x : a) {
            int l = 0, r = len;
            // 尝试在 a[i] 中寻找比 q[len-1] 大的数
            while (l < r) {
                int mid = (l + r + 1)>> 1;
                if (q[mid] < x) l = mid;
                else r = mid - 1;
            }
            // 在末尾更新
            len = max(len, r + 1);
            q[r+1] = x;
        }
        // len 为最长公共子序列的长度，要插入数据为 target.size() - len
        return target.size() - len;
    }
};

复杂度分析

时间复杂度：$O(NlogN)$

空间复杂度：$O(N)$
