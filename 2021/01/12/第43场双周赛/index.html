第43场双周赛

  2021-01-12   力扣 算法 周赛   贪心 数学 回溯 递归 树 图 0  评论 字数统计: 953字 阅读时长: 2分

题单：

  • 计算力扣银行的钱
  • 删除子字符串的最大得分
  • 构建字典序最大的可行序列
  • 重构一棵树的方案数

T1：计算力扣银行的钱

思路：

第一周：1, 2, 3, 4, 5, 6, 7

第二周：2, 3, 4, 5, 6, 7, 8

….

根据上述思路模拟即可。

代码

class Solution {
    public int totalMoney(int n) {
        int res = 0;
        // 当前日期   计数      上周开头的钱
        int day = 1, idx = 1, pre = 1;
        while (idx <= n) {
            res += day++;
            // 说明要开始下一周了
            if (idx >= 7 && idx % 7 == 0) {
                day = ++pre;
            }
            idx++;   
        }
        return res;
    }
}

复杂度分析

时间复杂度：$O(1)$

空间复杂度：$O(N)$

T2：删除子字符串的最大得分

思路：

我们从字符串每删除一个 ab 子串，就可以得到 x 分。每删除一个 ba 子串，就可以得到 y 分。

我们对字符串进行预处理，使得删除 ab 的得分是恒大于 ba 的得分。

那么对于一段连续的仅含有 ab 的子字符串，我们要做的操作就是尽可能删除所有的ab 和 ba对 ，由于我们一定可以删除完毕(恰好删完，或者仅剩一个)，所以我们可以得到：$c * x + (min(a, b) - c) * y$。其中，$c$ 为删除 ab 的操作次数，对应的剩下的$min(a,b)-c$ 就是删除 ba的操作次数。

代码

class Solution {
public:
    int maximumGain(string s, int x, int y) {
        // 预处理
        if (x < y) {
            swap(x, y);
            for (auto &c : s) {
                if (c == 'a') c = 'b';
                else if (c == 'b') c = 'a';
            }
        }
        int res = 0;
        for (int i = 0; i < s.size(); i++) {
            if (s[i] != 'a' && s[i] != 'b') continue;
            int j = i + 1;
            while (j < s.size() && (s[j] == 'a' || s[j] == 'b')) j++;
            // 区间 [i,j] 为仅含 a，b的字符串
            int a = 0, b = 0, c = 0, t = 0;
            for (int k = j - 1; k >= i; k--) {
                if (s[k] == 'a') {
                    a ++;
                    if (t) {
                        t--;
                        c++;
                    }
                } else {
                    b ++;
                    t ++;
                }
            }
            // 计算这段字符串可以产生的得分
            res += x * c + (min(a, b) - c) * y;
            i = j - 1;
        }
        return res;
    }
};

复杂度分析

时间复杂度：$O(N^2)$，其中最差的情况是，字符串中仅含a，b

空间复杂度：$O(1)$

T3：构建字典序最大的可行序列

思路

我们需要构造一个序列

  • 1 只出现过一次
  • [2, n] 每个数是出现 两次
  • [2, n]每个数的出现位置间隔为 i

由于数据范围很小 $1 \le n \le 20$ ，所以我们可以直接暴力枚举

代码

class Solution {
public:

    int n;
    vector<bool> st;
    vector<int> path;

    bool dfs(int u) {
        if (u == 2*n - 1) return true;
        if (path[u]) return dfs(u + 1);
        // 从最大的数开始，要求字典序最大
        for (int i = n ; i > 1; i--) {
            if (!st[i] && u + i < n*2-1 && !path[u + i]) {
                st[i] = true;
                path[u] = path[u + i] = i;
                if (dfs(u + 1)) return true;
                st[i] = false;
                path[u] = path[u + i] = 0;
            }
        }
        if (!st[1]) {
            st[1] = true;
            path[u] =  1;
            if (dfs(u + 1)) return true;
            st[1] = false;
            path[u] = 0;
        }
        return false;
    }


    vector<int> constructDistancedSequence(int _n) {
        n = _n;
        st.resize(n + 1, false);
        path.resize(n * 2 - 1);
        dfs(0);
        return path;
    }
};

复杂度分析

时间复杂度：$O(N!)$，其中每个位置，我们都要至少检查 $N$ 次

空间复杂度：$O(N)$
