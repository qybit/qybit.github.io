第224场周赛

  2021-01-17   力扣 算法 周赛   贪心 数组 排序 0 0  评论 字数统计: 818字 阅读时长: 2分

题单：

  • 可以形成最大正方形的矩形数目
  • 同积元组
  • 重新排列后的最大子矩阵
  • 猫和老鼠 II

T1：可以形成最大正方形的矩形数目

思路：

对于 $v[i] = min(rectangles[i][0], rectangles[i][1]$ ，我们需要记录 $maxLen = max{v[i]}$，然后再统计 $v[i] \ge maxLen$ 的个数。

代码：

class Solution {
public:
    int countGoodRectangles(vector<vector<int>>& r) {
        int n = r.size();
        vector<int> v(n);
        int ma = 0;
        for(int i = 0; i < n; i ++) v[i]=min(r[i][0],r[i][1]),ma=max(ma,v[i]);
        int res = 0;
        for (auto &x : v) if (x >= ma) res++;
        return res;
    }
};

复杂度分析：

时间复杂度：$O(N)$

空间复杂度：$O(N)$

T2：同积元组

思路：

根据题意中 $nums = [2,3,4,6]$，可知 $(a,b,c,d)$ 为一个元组，假设 $(a,b)=[3,4], (c,d)=[2,6]$

可以发现，扩展一下 $(b,a)$ 和 $(d,c)$ ，每一对内部互换的话，仍可以满足要求。$(c,d)$ 和 $(a,b)$ 互换也是符合要求的，所以每一次互换提供 2 个满足的元组，那么一对 $(a,b,c,d)$ 可以贡献 $2^3$ 个元组。

我们只需要记录下， $a*b，a \ne b$ 的元素个数即可。

代码：

class Solution {
public:
    int tupleSameProduct(vector<int>& nums) {
        unordered_map<int,int> hash;
        int res = 0, n = nums.size();
        for (int i = 0; i < n; i ++) {
            for (int j  = i+1; j < n; j++) {
                res += 8 * hash[nums[i]*nums[j]];
                hash[nums[i] * nums[j]]++;
            }
        }
        return res;
    }
};

复杂度分析：

时间复杂度：$O(N^2)$

空间复杂度：$O(N^2)$

T3：重新排列后的最大子矩阵

思路：

定义：$matrix[i][j]$ 为以 $(i,j)$ 为结尾向上连续1的最大长度。

对于这个问题，我们可以问题转化为，求柱状图的最大面积来做。

具体做法：

我们需要预处理原数组，计算出每一行的"高"，即在点 $(i,j)$ 处，以 1 结尾的可以向上连续1的最大长度。

由于我们可以按照 列 任意顺序重新排列，那么我们对"高"进行排序，从最高点开始枚举。

为了方便理解，假设一个情况下，预处理完毕后的某一行数组的高度如下：

由于我们任意的交换列，所以我们可以对进行降序排序，然后再计算最大面积

代码：

class Solution {
public:
    int largestSubmatrix(vector<vector<int>>& w) {
        int n = w.size(), m = w[0].size();
        for (int i = 1; i < n; i ++) 
            for (int j = 0; j < m; j++) 
                if (w[i][j]) w[i][j]+= w[i-1][j];
        
        int res = 0;
        vector<int> q(m);
        for (int i = 0; i < n; i ++) {
            for (int j = 0; j < m; j++) q[j] = w[i][j];
            // 降序排序
            sort(q.begin(), q.end(), greater<int>());
            for (int j = 0; j < m; j++) {
                // 枚举所有的高 * 宽
                res = max(res, q[j] * (j+1));
            }
        }
        return res;
    }
};

复杂度分析

时间复杂度：$O(N * MlogM)$

空间复杂度：$O(1)$
