Trie树

时间： 2020-11-26   |   分类： 数据结构   | 字数： 1323 字 | 阅读： 3分钟 | 阅读次数：

Trie树是一种特殊的数据结构，它常用于在数据中检索字符串。也可以叫做字典树和前缀树等等。

一、应用

1、搜索引擎的搜索推荐

2、文档和文本检索功能

3、打字预测

二、数据结构

检索字符串的操作中，我们大可以选择像Set, HashMap等高级数据结构，但是当数据量很大时，不同数据计算出的hash可能会存在重复的情况，导致时间复杂度不再是理想状态的$O(1)$，而是使查询效率退化至$O(n)$ ，n为查询次数（具体可以查看HashMap解决hash冲突的方法）。而且如果键长到一定情况时，查询任一字符串时都是$O(n)$效率的话，用HashMap的效率会显得很不尽人意。所以HashMap的缺点就显现出来了。

  • 无法应对大量的查询任务，存在hash冲突。
  • 按字典序的多次查询任务。

字典树的查询效率为$O(m)$，m为字符串的长度。

结构特点

字典树是一个有根节点的树：

  • 子节点是字母表中的所有的元素，本文的字母表仅限于26个小写字母。
  • 布尔值，可以确定当前字母是否为一个完整的键。

TrieNode:

private static class TrieNode{
    private boolean isEnd;
    private final TireNode[] next; 
    private static final int R = 26;

    public TrieNode(){
        next = new TrieNode[R];
    }

    public boolean containsKey(char c) {
        return next[c-'a'] != null;
    }
    public TireNode get(char c) {
        return next[c-'a'];
    }

    public void insert(char c, TrieNode node) {
        next[c-'a'] = node;
    }

    public boolean isEnd() {
        return isEnd;
    }

    public void setEnd() {
        isEnd = true;
    }
}

字典树的最常用的方法就是插入和查找。

3、方法实现

1、insert

在树中插入一个新单词。现在树中存在的单词可能为$[“apply”, “a”, “abc”, “ab”]$

现插入新单词 abandon后。

具体操作：

插入之前，先找到其根节点，从根节点出发找到第一个字母对应的节点，对于下一个字母有两种情况需要考虑：

  • 存在通路。即已存在节点，进行移动到下一层。
  • 不存在通路。即不存在节点，创建一个新节点，然后移动到新节点

重复这两步操作，直到这个单词被遍历完毕。当到达单词最后一个，将最后一个节点设置为结束标志。

复杂度分析：

时间复杂度：$O(m)$，m为字符串的长度，每一层在选择节点时的操作是$O(1)$，需要把所有的字母都走一遍。

空间复杂度：$O(m)$，m为字符串的长度，最差的情况需要创建m个节点，最好的情况是不需要创建新节点。

2、search

在字典树中搜索字符。先字典树中存在单词有$[“apply”, “a”, “abc”, “ab”, “abandon”]$

例如，在字典树中搜索单词abandon

具体操作：

对于待搜索的一个单词，我们先从根节点开始搜索，从单词的第一个字母开始查找节点，那么对于任意节点都有两种情况：

  • 存在通路。即已存在节点，移动到下一层。
  • 不存在通路。说明节点不存在，那么搜索失败。树中并不存在此节点

重复上述两种操作，直到读取结束标志时，才算是一次成功的搜索，即说明字符串已经存在。

复杂度分析：

时间复杂度：$O(m)$，m为字符串长度，最差为就是搜索成功的时候，即所有字母全部检索到。

空间复杂度：$O(1)$

4、完整代码实现

public class Trie {

    private static class TrieNode {
        private boolean isEnd;
        private final TrieNode[] next;
        private static final int R = 26;

        public TrieNode(){
            next = new TrieNode[R];
        }

        public boolean containsKey(char c) {
            return next[c-'a'] != null;
        }
        public TrieNode get(char c) {
            return next[c-'a'];
        }

        public void put(char c, TrieNode node) {
            next[c-'a'] = node;
        }

        public boolean isEnd() {
            return isEnd;
        }

        public void setEnd() {
            isEnd = false;
        }
    }

    private final TrieNode root;

    public Trie() {
        root = new TrieNode();
    }

    public void insert(String word) {
        TrieNode node = root;
        for (char c : word.toCharArray()) {
            if (!node.containsKey(c)) {
                node.put(c, new TrieNode());
            }
            node = node.get(c);
        }
        node.setEnd();
    }

    public boolean search(String word) {
        TrieNode node = root;
        for (char c : word.toCharArray()) {
            node = node.get(c);
            if (node == null) {
                return false;
            }
        }
        return node.isEnd();
    }

    public boolean startsWith(String prefix) {
        TrieNode node = root;
        for (char c : prefix.toCharArray()) {
            node = node.get(c);
            if (node == null) {
                return false;
            }
        }
        return true;
    }
}
