第40场双周赛

时间： 2020-11-29   |   分类： 力扣   算法   周赛   | 字数： 1096 字 | 阅读： 3分钟 | 阅读次数：

难度不大，很常规的题目。最后一题动态规划仍需要多练。

T1：最大重复子字符串

思路：

  • 计算在主串中，可以将字串word重复的最大次数

java:

class Solution {
    public int maxRepeating(String sequence, String word) {
        int k = 0;
        StringBuilder s = new StringBuilder(word);
        while (sequence.contains(s.toString())) {
            k++;
            s.append(word);
        }
        return k;
    }
}

python:

class Solution:
    def maxRepeating(self, sequence: str, word: str) -> int:
        k = 0
        s = word
        while s in sequence:
            k += 1
            s += word
        return k

复杂度分析：

时间复杂度：$O(n*m)$ 其中n为主串的长度，m为字串的长度

空间复杂度：$O(m)$ 其中m为字串的长度

T2：合并两个链表

思路：

  • 关于链表删除，最好的建议是引入一个虚拟头节点，来预防头节点的删除。
  • 此题只需要遍历a-1个节点，然后把list2遍历至最后一个节点，再继续遍历list1至第b个节点。然后把a-1链接至list2的头节点，list2的尾节点，连接至b+1个节点。
  • 模拟上述过程即可。
class Solution {
    public ListNode mergeInBetween(ListNode list1, int a, int b, ListNode list2) {
        ListNode dummy = new ListNode(-1);
        dummy.next = list1;
        int idx = 0;
        ListNode cur = dummy;
        while (idx < a) {
            idx++;
            cur = cur.next;
        }
        ListNode c1 = cur;
        while (idx <= b) {
            idx++;
            cur = cur.next;
        }
        ListNode c2 = cur.next;
        ListNode c3 = list2;
        while (c3.next != null) {
            c3 = c3.next;
        }
        c3.next = c2;
        c1.next = list2;
        return list1;
    }
}

复杂度分析：

时间复杂度：$O(n)$

空间复杂度：$O(1)$

T3：设计前中后队列

思路：

  • 利用列表模拟队列。
  • 对于中间位置，当队列为奇数个元素时，弹出中间位置的前一个。当队列为偶数个元素时，仍弹出前一个元素。因此借助ArrayList的插入操作可以解决问题。
  • 对于中间元素，需要特别注意的是，中间元素-1。
class FrontMiddleBackQueue {
    
    List<Integer> list = new ArrayList<>(5000);

    public FrontMiddleBackQueue() {

    }
    
    public void pushFront(int val) {
        if (!list.isEmpty()) {
            list.add(0, val);
        } else {
            list.add(val);
        }
    }
    
    public void pushMiddle(int val) {
        if (list.isEmpty()) {
            list.add(val);
        } else {
            int sz = list.size();
            if (sz % 2 != 0) sz--;
            int t = sz>>1;
            list.add(t, val);
        }
     }
    
    public void pushBack(int val) {
        list.add(val);
    }
    
    public int popFront() {
        if (!list.isEmpty()) {
            return list.remove(0);
        }
        return -1;
    }
    
    public int popMiddle() {
        if (!list.isEmpty()) {
            int sz = list.size()-1;
            int t = sz>>1;
            return list.remove(t);
        }
        return -1;
    }
    
    public int popBack() {
        if (!list.isEmpty()) {
            return list.remove(list.size()-1);
        }
        return -1;
    }
}

复杂度分析：

时间复杂度：$O(n)$，最好$O(1)$，最差$O(n)$。其中n为列表的长度

空间复杂度：$O(n)$

T4：得到山形数组的最少删除次数

思路：

  • 根据山形状数组的特性可以发现，从0开始不断上升至最大值，然后从最大值开始，逐渐下降至最小值
  • 那么我们的思路就可以同时寻找最长的上升子序列和最长的下降子序列，取两者相交的最大值。就可以得到最长的山形状数组的长度，然后拿原数组的长度减去山形数组，即表示为要删除的次数
class Solution {
    public int minimumMountainRemovals(int[] nums) {
        int n = nums.length;
        int[] f = new int[n]; // 最长升序子序列
        int[] g = new int[n]; // 最长降序子序列
        for (int i = 0; i < n; i++) {
            f[i] = g[i] = 1;
        }
        for (int i = 1; i < n; i++) {
            for (int j = 0; j < i; j++) {
                if (nums[i] > nums[j]) {
                    f[i] = Math.max(f[i], f[j]+1);
                }
            }
        }
        
        for (int i = n-2; i >= 0; i--) {
            for (int j = i+1; j < n; j++) {
                if (nums[i] > nums[j]) {
                    g[i] = Math.max(g[i], g[j]+1);
                }
            }
        }
        int res = 0;
        for (int i = 0; i < n; i++) {
            if (f[i] > 1 && g[i] > 1) {
                res = Math.max(res, g[i]+f[i]-1);
            }
        }
        // res 为可以得到的最长的山形数组的长度，原数组长度-山形数组长度 = 要删除的次数
        return n - res;
    }
}

复杂度分析：

时间复杂度：$O(n^2)$

空间复杂度：$O(n)$
