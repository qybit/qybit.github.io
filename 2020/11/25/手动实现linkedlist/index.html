手动实现LinkedList

时间： 2020-11-25   |   分类： 数据结构   | 字数： 1816 字 | 阅读： 4分钟 | 阅读次数：

前言

链表是与线性表相对应的一种特殊的数据结构，它的各个节点在物理内存上可以是连续的也可以是不连续的，逻辑上链表是连续的。

优点

  • 增删效率高
  • 适合经常更改的应用场景

缺点

  • 查询效率很低
  • 不适合经常查询的应用场景

优缺点是相当于线性表而言！

官方的提供的LinkedList就是链表实现的，双向链表数据结构。主要功能有，头插法，尾插法，头删除，尾删除等常用的操作。一般在应用中的体现就是，常用LinkedList来作为队列，甚至是栈等特殊需求。

核心功能：

  • offerFirst(val) / offerLast(val)
  • pollFirst() / pollLast()
  • peekFirst() / peekLast()

基本使用：

LinkedList<Integer> list = new LinkedList<>();

list.offerLast(1); // 1
list.offerLast(2); // 1 -> 2

list.pollFirst(); // 2

list.peekLast(); // return 2

功能实现：

核心功能原理实现

1、成员变量

// 链表每个节点
private static class Node {
    int val; // 节点值
    Node next; // 后继节点
    Node prev; // 前驱节点

    public Node(int v) { // 构造函数
        val = v;
    }
}

Node first; // 头节点
Node last; // 尾节点
int size; // 链表长度

2、offerFirst(val)

该方法是在链表的头节点，插入一个新的节点。

  • 先判断，头节点是否为空。如果为空，则说明链表此时为空。需要将首尾节点都赋值为新节点。
  • 通过头节点进行操作
    • newNode的next指针指向 first
    • first的前驱指向 newNode
    • 再将头指针移动到newNode

3、offerLast(val)

该方法是在链表的尾部，加入一个新的节点。

  • 在插入时，仍需要考虑尾节点是否为空。如果为空，则说明链表此时为空。需要将首尾节点都赋值为新节点。
  • 通过尾节点进行操作：
    • newNode的prev指针指向last
    • last的next指针指向newNode
    • 再将last指针指向newNode

4、pollLast

移除链表的最后一个元素。

  • 首先会检查，链表是否为空。如果空，直接抛出异常。
  • last指针前移，然后将next置为空。

5、pollFirst

移除链表的头一个元素。

  • 首先会检查，链表是否为空。如果空，直接抛出异常。
  • first指针后移，然后将prev置为空。

6、peekLast / peekFirst

在头尾节点不为空的情况，返回值val即可。

7、get(index)

该方法表示获取index位置的节点。

  • $0 \le index <= size-1$

扩展

  • addAtIndex(index, val) 表示可以在范围 $0 \le index \le size-1$ 上的任意位置之前，添加节点。
  • deleteAtIndex(index)表示在范围$0 \le index \le size-1$上的任意位置的一个节点。

1、addAtIndex(index, val)

根据index进行分类讨论

  • 检查越界，即抛出异常。

  • index == 0成立，即头插法。

  • index==size-1成立，即尾插法。

  • 在$[1, size-1)$范围上时

    • 伪代码如下（类python代码）：

      cur = get(index=1) // 索引对应的节点
newNode = Node(val)
// 对应序号1
prev = cur.prev	// 前驱节点
prev.next = newNode
// 对应序号2
prev.next = newNode
newNode.prev = prev
// 对应序号3
newNode.next = cur
cur.prev = newNode
size += 1

      ​

2、deleteAtIndex(index)

同样的根据index进行讨论：

  • 越界检查，如果越界抛出异常。

  • index==0移除头节点

  • index==size-1移除尾节点

  • 在$[1, size-1)$时

    • 伪代码：

      cur = get(index=1)
// 获得前驱节点和后继节点
prev = cur.prev
next = cur.next
// 前驱节点的后继指向后继的后继
// 后继节点的前驱指向前驱的前驱
prev.next = next
next.prev = prev

完整代码

/**
 * @author qybit
 */
public class MyLinkedList {

    private static class Node {
        int val;
        Node next;
        Node prev;

        public Node(int v) {
            val = v;
        }
    }

    Node first;
    Node last;
    int size;


    public MyLinkedList() {

    }

    /**
     * @param index [0, size-1]
     * @return 返回index处节点的值
     */
    public int get(int index) {
        if (index < 0 || index >= size) {
            throw new RuntimeException("the index out of the limited");
        }
        return node(index).val;
    }

    /**
     * 如果index > size/2的话，此时从尾部开始搜索
     * 否则就从头部开始搜索
     *
     * @param index [0, size]
     * @return Node
     */
    public Node node(int index) {
        // 从尾部开始搜索
        Node x;
        if (index > (size >> 1)) {
            x = last;
            for (int i = size - 1; i > index; i--) {
                x = x.prev;
            }
        } else {
            x = first;
            for (int i = 0; i < index; i++) {
                x = x.next;
            }
        }
        return x;
    }

    /**
     * 在头部插入值
     *
     * @param val 待插入的值
     */
    public void offerFirst(int val) {
        Node node = new Node(val);
        if (first == null) {
            last = node;
        } else {
            node.next = first;
            first.prev = node;
        }
        first = node;
        size++;
    }

    /**
     * 在尾部插入值
     *
     * @param val 待插入的值
     */
    public void offerLast(int val) {
        Node node = new Node(val);
        if (last == null) {
            first = node;
        } else {
            last.next = node;
            node.prev = last;
        }
        last = node;
        size++;
    }

    /**
     * 在任意索引之前，加入节点
     *
     * @param index [0, size-1] 范围
     * @param val   待插入节点的值
     */
    public void addAtIndex(int index, int val) {
        if (index > size) {
            throw new RuntimeException("the index out of the limited");
        }
        if (index <= 0) {
            offerFirst(val);
            return;
        }
        if (index == size) {
            offerLast(val);
            return;
        }
        // [1, size-1]
        Node origin = node(index);
        Node node = new Node(val);
        Node prev = origin.prev;
        origin.prev = node;
        node.next = origin;
        prev.next = node;
        node.prev = prev;
        size++;
    }

    /**
     * 删除指定范围内的节点
     *
     * @param index 范围[0, size-1]
     */
    public void deleteAtIndex(int index) {
        if (index < 0 || index >= size) {
            throw new RuntimeException("the index out of the limited");
        }
        if (index == 0) {
            removeHead();
            return;
        }
        if (index == size - 1) {
            removeLast();
            return;
        }
        // [1, size-1]
        Node node = node(index);
        Node prev = node.prev;
        Node next = node.next;
        prev.next = next;
        next.prev = prev;
        // 将该节点置空
        node.val = -1;
        node.next = null;
        node.prev = null;
        size--;
    }

    /**
     * 移除尾部节点
     */
    private void removeLast() {
        Node l = last;
        Node prev = l.prev;
        last = prev;
        if (last == null) {
            first = null;
        } else {
            prev.next = null;
        }
        size--;
    }

    /**
     * 移除头部节点
     */
    private void removeHead() {
        Node f = first;
        Node next = f.next;
        first = next;
        if (first == null) {
            last = null;
        } else {
            next.prev = null;
        }
        size--;
    }
}
