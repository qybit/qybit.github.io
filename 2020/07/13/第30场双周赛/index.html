第30场双周赛

  2020-07-13   周赛 力扣 算法   字符串 数组 排序 动态规划 力扣 周赛 0  评论 字数统计: 969字 阅读时长: 2分

这场比赛的题目不是很难，即使第一次接触的话，AC前两题应该也是没有问题的。废话不多说，看代码。

题目一：转变日期格式

这个题，我是不知道有什么API可以调用，但是看它给出的YYYY-MM-DD的格式，其实稍微想想还是硬解析的。无脑暴力即可。注意日期和月份小于10时，需要补0；

class Solution {
    public String reformatDate(String date) {
        String[] months = {"Jan", "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"};
        StringBuilder sb = new StringBuilder();
        String[] data = date.split(" ");
        // 看格式 按空格划分
        String year = data[2], month = data[1], day = data[0];
        // 年份直接加入
        sb.append(year).append("-");
        // 月份查表
        for (int i = 0; i < months.length; i++) {
            if (months[i].equals(month)) {
                int m = i+1;
                if (m < 10) { // 小于10， 要补0
                    sb.append(0);
                }
                sb.append(m).append("-");
            }
        }
        // 找第一个字母的下标
        char[] days = day.toCharArray();
        int idx = 0;
        for (int i = 0; i < days.length; i++) {
            if (!Character.isDigit(days[i])) {
                idx = i;
                break;
            }
        }
        // 找到下标，转数字
        int d = Integer.parseInt(data[0].substring(0, idx));
        if (d < 10) { // 小于10 补0
            sb.append(0);
        }
        sb.append(d);
        
        return sb.toString();
    }
}

时间复杂度：$O(n)$

题目二：子数组和排序后的区间和

按照题目要求即可。要注意给出的区间 left和right。在数组的范围其实是[left-1, right-1]。

class Solution {
    public int rangeSum(int[] nums, int n, int left, int right) {
        List<Integer> arr = new ArrayList<>();
        // 模拟
        for (int i = 0; i < nums.length; i++) {
            int sum = 0;
            for (int j = i; j < nums.length; j++) {
                sum += nums[j];
                arr.add(sum);
            }
        }
        // 排序
        Collections.sort(arr);
        // 求区间和，  [left-1, right-1]
        int res = 0;
        for (int i = left - 1; i < right; i++) {
            res = (res + arr.get(i)) % 1000000007;
        }
        return res;
    }
}

时间复杂度：$O(n^2\log n)$

题目三：三次操作后最大值于最小值的最小差

这是一个脑经急转弯题。最大值于最小值的差，实际上于中间值并没有关系。我们排序后，其实找最大值和最小值，进行比较即可。

class Solution {
    public int minDifference(int[] nums) {
        Arrays.sort(nums);
        int n = nums.length;
        // 由于我们只能修改三次，所以要么修改最大值，要么修改最小值
        if (n < 5) {
            return 0;
        }
        int a = Math.min(nums[n-1] - nums[3], nums[n-2] - nums[2]);
        int b = Math.min(nums[n-3] - nums[1], nums[n-4] - nums[0]);
        return Math.min(a, b);
    }
}

时间复杂度：$O(n\log n)$

题目四：石子游戏 IV

这是一个博弈DP问题。对于任意个 $ 1 \le i \le n$ ，我们从 $ 1 \le j \le \sqrt i $ ，列举二人拿石子的输赢情况。

class Solution {
    public boolean winnerSquareGame(int n) {
        // dp[i] 表示有i个石子，先手是否可以赢
        boolean[] dp = new boolean[n+1];
        Arrays.fill(dp, false);
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j * j <= i; j++) {
                if (!dp[i -j*j]) {// 假设Alice选完平方数个石子后，对手不能赢，那么就是Alice赢了
                    dp[i] = true;
                }
            }
        }
        return dp[n];
    }
}

时间复杂度：$O(n^2)$

对应的给出记忆化DFS搜索代码：

class Solution {
    Map<Integer, Boolean> map = new HashMap<>();
    public boolean winnerSquareGame(int n) {
       return dfs(n);
    }
    
    private boolean dfs(int n) {
        if (n == 0) return false;
        if (n == 1) return true;
        if (map.containsKey(n)) return map.get(n);
        int i = 1;
        while (i*i <= n) {
            if (!dfs(n- i*i)) { // 枚举出所有 拿平方数的情况，假设 bob不能赢，那么Alice就赢了
                map.put(n, true);
                return true;
            }
            i++;
        }
        // 记忆化结果
        map.put(n, false);
        return false;
    }
}

时间复杂度：$O(n^2 \log n)$
