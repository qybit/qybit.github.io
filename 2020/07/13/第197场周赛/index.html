题目一：好数对的数目

题解：略

暴力法：

class Solution {
    public int numIdenticalPairs(int[] nums) {
        int res = 0;
        for (int i = 0; i < nums.length; i++) {
            for (int j = i+1; j < nums.length; j++) {
                if (i < j && nums[i] == nums[j]) res++;
            }
        }
        return res;
    }
}

时间复杂度：$O(n^2)$

如果数据量很大的话，就要考虑空间换时间了。

优化：

先计数后，两个相同的数，下标一定是不相同的。所有我们只需要统计 i < j 的情况，即对于某个数 num ，好数对存在 ${2 \choose n}$ 个。

class Solution {
    public int numIdenticalPairs(int[] nums) {
        Map<Integer, Integer> map = new HashMap<>();
        for (int num : nums) {
            map.put(num, map.getOrDefault(num, 0) + 1);
        }
        int res = 0;
        for (int key : map.keySet()) {
            int cnt = map.get(key);
            res += cnt * (cnt-1) >> 1;
        }
        return res;
    }
}

时间复杂度：$ O(n) $

题目二：仅含1的字串数

首先暴力回溯肯定是超时的解法。因为时间复杂度达到了 $ O(n^{n}) $

我们观察用例，可以发现当连续的字串数量为 3 时，它贡献的字串数量为 $ 3 * (3 + 1) \over 2 $ ，由此可以推广至当连续的字串数量为 n 时，它贡献的字串数量为 $ n * (n + 1) \over 2 $

代码：

注意点：类型溢出，int 肯定是装不下的

class Solution {
    public int numSub(String s) {
        long mod = 1000000007;
        long cur = 0, res = 0;
        char[] chars = s.toCharArray();
        for (char c : chars) {
            if (c == '1') {
                cur++;
            } else {  
                res += cur * (cur+1) >> 1;
                res %= mod;
                cur = 0;
            }
            
        }
        // 以 1 结尾的字串
        res += cur * (cur+1) >> 1;
        res %= mod;
        return (int)res;
    }
}

题目三：概率最大的路径

大师，我终于悟了！其实这个题是比较基本的迪杰特斯拉算法，在遍历所有的路径，加权衡量要遍历哪个节点时，让概率最大的节点在对头。在BFS的过程中，加入贪心的思想，每次总是走概率最大的那条路。

class Solution {
    
    class Node{
        int id;
        double weight;
        Node() {
            
        }
        
        Node(int _id, double _weight) {
            id = _id;
            weight = _weight;
        }
    }
    
    public double maxProbability(int n, int[][] edges, double[] succProb, int start, int end) {
        List<Node>[] graph = new ArrayList[n];
        // 初始化
        for (int i = 0; i < n; i++) {
            graph[i] = new ArrayList<>();
        }
        // 建图
        for (int i = 0; i < edges.length; i++) {
            int from = edges[i][0];
            int to = edges[i][1];
            double weight = succProb[i];
            graph[from].add(new Node(to, weight));
            graph[to].add(new Node(from, weight));
        }
        // 用一个大根堆代替普通的队列，让概念大的总是在对头
        PriorityQueue<Node> queue = new PriorityQueue<>((a, b) -> Double.compare(b.weight, a.weight));
        boolean[] visited = new boolean[n];
        queue.offer(new Node(start, 1));
        visited[start] = true;
        while (!queue.isEmpty()) {
            Node node = queue.poll();
            if (node.id == end) return node.weight;
            visited[node.id] = true;
            for (Node ne : graph[node.id]) {
                if (!visited[ne.id]) {
                    Node next = new Node(ne.id, ne.weight*node.weight);
                    queue.offer(next);
                }
            }
        }
        return 0D;
    }
    
}
